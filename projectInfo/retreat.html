ReTreat is a simple, Twitter-like social media platform.

The website is hosted on Cloudflare Pages and uses Cloudflare Workers for the backend.
It has an intuitive authentication screen, combining a log-in and sign-up screen into one, coherent system.
It has image support, in the form of Base-64 encoded images, and a simple, easy to use interface.
