A clean, terminal based text editor with some Vim-like keybinds.
It is programmed in C, and efficiently allocates and de-allocates memory to ensure a smooth and responsive experience.
It supports the simple Vim keybinds, and entering Insert Mode allows for normal movement with arrow keys.
