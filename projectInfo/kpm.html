Kindle Package Manager ("kpm") is the first freely available package manager for kindles.

It has inspired other projects such as KindleForge, a graphical package manager, with its simplicity and ease of use.
kpm is purely written in 800 or so lines of C, 90% of which written by my own hand.
It has gathered so much attention that it even has it's own wiki, created by myself and Kurizu
